# Ultimate Mortal Kombat: Shaolin Monks — Fan Site Redesign

A complete visual overhaul of the original mod page by **RelaxDirk**.  
All content, characters, and move data belong to him.

## What's Included

- `umksm.html` — the full site, one single file, no dependencies

## Features

- All characters with move lists, combos, specials, and finishers
- Filter by Story / Boss / Enemy / Secret
- Click any character portrait to jump to their move card
- Cheat code section
- Credits section
- Mobile friendly
- Works offline — just open the HTML file in any browser

## How to Use

Just open `umksm.html` in a browser. No server, no install, nothing else needed.

To host it (e.g. on GitHub Pages), upload the file and point the repo to it.
