## Controller Layout Reference

### Xbox Controller
| Action | Button |
|--------|--------|
| Quick Attack | `X` |
| Launching Attacks | `Y` |
| Power Attacks | `B` |
| Jump/Evade | `A` |
| Throws/Pick-up Weapons | `BLACK` |
| Fatality Starter | `WHITE` |
| Lock On/Block | `LT` |
| Modifier | `RT` |
| Enemy Manipulation | `RT + BLACK` |

### PlayStation 2 Controller
| Action | Button |
|--------|--------|
| Quick Attack | `□` (Square) |
| Launching Attacks | `△` (Triangle) |
| Power Attacks | `○` (Circle) |
| Jump/Evade | `X` |
| Throws/Pick-up Weapons | `R2` |
| Fatality Starter | `L1` |
| Lock On/Block | `L2` |
| Modifier | `R1` |
| Enemy Manipulation | `R1 + R2` |

### Directional Notation
- **Up** = ↑
- **Down** = ↓
- **Left** = ←
- **Right** = →

## Notes

- **EXP Requirements**: Some moves require experience points to unlock. The EXP values shown indicate when the move becomes available.
- **Directional Inputs**: Fatality, Multality, and Brutality inputs are the same for both controllers; only the final button press differs.
- **Hold Times**: Some special moves require holding buttons for specific durations (2 or 3 seconds).
- **In-Air Moves**: Certain moves can only be performed while jumping/in the air.
- **Enemy Manipulation**: Available for some characters, allows controlling defeated enemies temporarily.
